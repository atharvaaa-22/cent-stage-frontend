## Setup
1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_API_URL` (default http://localhost:4000)
2. Install:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

Open http://localhost:3000

- Uses React Query for fetching
